WP Twitch Status is a free plugin for WordPress that uses the API from Twitch to retrieve the data of a specified Twitch stream to show its status on your website.

== Description ==

WP Twitch Status is a free plugin for WordPress that uses the API from Twitch to retrieve the data of a specified Twitch stream to show its status on your website.

You can configure it by going to Settings > General > WP Twitch Status and filling in all the details. The fields are all required. Click on the links “Learn more” in the description of the fields to learn how to get the necessary data.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of WP Twitch Status, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “WP Twitch Status” and click Search Plugins. Once you’ve found the plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading this plugin plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
